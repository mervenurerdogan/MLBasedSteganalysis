# MLBasedSteganalysis

Bu çalışmada BASEbose ve BASEbose_lsb veri setleri kullanılarak makine öğrenmesi algoritmaları ile steganaliz yapılmıştır.
Kullanılan algoritmalar sırasıyla
* Support Vector Machine(SVM)
* KNN
*Random Forest
*Decision Tree
*Bagging classifier
